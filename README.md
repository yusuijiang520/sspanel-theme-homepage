## 安装脚本
``` bash
wget -N --no-check-certificate https://raw.githubusercontent.com/lizhongnian/sspanel-theme-homepage/master/install.sh &&
chmod +x install.sh &&
bash install.sh
```
复制执行以上脚本，如下图：


第一步，输入网站目录+模板编号</br>
第二步，根据返回的信息核对你输入的是否正确</br>
第三步，如果正确按1继续，不正确按2退出</br>
第四步，然后就静静地等待自动安装，安装完成会输入成功的提示</br>
